* 시간
종료 시간: 1104
시작 시간: 1013
소요 시간: 51분

* 주제:
카드놀이

* 설명:
두 개의 카드 세트를 가지고 있음
각각 1부터 N, 1부터 M 까지의 카드로 구성됨
각 카드 세트에서 각각 한 개의 카드를 뽑았을 때, 그 합이 가장 많이 나올것으로 예상되는 수를 구하라.
카드를 선택할 확률은 동일하다고 가정한다.

* 제약 사항:
카드 세트에서 임의의 카드를 뽑을 확률은 모두 동일하다.
N과 M은 4이상 20이하의 정수이다.

* 입력:
테스트케이스 개수
N1 M1
N2 M2
...

* 출력:
#1 00
#2 00

* 풀이:
일단 생각나는 풀이는 N과 M의 합 - 1만큼의 배열을 생성한다.
1부터 N까지 M과 더한 후 배열의 값을 증가시킨다.
등장할 확률이 높은 수를 오름차순으로 출력한다.

N: 1, M: 2
N: 0    // 1
M: 0 1  // 1, 2
N + M : 0, 1    // 2, 3
