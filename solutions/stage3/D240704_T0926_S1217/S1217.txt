재귀(){
  종료조건
  실행부

}
밑이 n이고 지수가 0이면 1을 반환
밑이 n이고 지수가 m이면 n^(m)을 반환

밑이 2이고 지수가 2인 경우
재귀(2, 1);

재귀(밑, 지수){
  만약 지수가 0이면
    밑 = 밑 * 1
  지수가 0보다 크면
    밑 = 밑 * 밑
    --지수
    재귀(밑, 지수)
}

밑과 지수
처음 삽입했을 때 지수가 0인 경우{
    리턴 1;
} 아닌 경우{
    값 = 밑 * 밑;
    -- 지수;
    재귀호출;
}
