* 시간
종료 시간: 1519
시작 시간: 1503
소요 시간: 16분

* 주제:
제로

* 설명:
입력 횟수만큼 숫자들이 입력될 때, 0 이 입력된 경우 이전에 입력된 숫자를 제외하라

* 제약 사항:
입력 횟수 K는 1 이상 100,000 이하이다
숫자는 0 이상 100,000 이하이다
정수가 0일 경우 지울 수 있는 수가 있음을 보장한다

* 입력:
테스트케이스
입력횟수1
숫자1
숫자2
...
입력횟수2
숫자1
숫자2
...

* 출력:
#1 00
#2 00

* 풀이:
스택을 이용해 입력되는 값을 저장해 놓는다.
숫자가 0인 경우 스택 탑을 제외한다.
입력이 완료되면 스택의 값들을 합산한다.
