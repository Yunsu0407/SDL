* 시간
종료 시간:
시작 시간: 2138
소요 시간:

* 주제:
마법사 상어와 토네이도

* 설명:
N*N 크기의 판에 (N/2, N/2)부터 좌,하,우,상으로 이동횟수를 가진채 이동한다.
이동횟수는 2회마다 1씩 증가하며 처음 이동횟수는 1이다.
이동은 (0,0)에 도착하는 순간 종료한다.

이동할 때마다 모래가 같이 이동하는데 모래의 이동량은 다음 5*5판 형태와 같다.
 0  0  2  0  0
 0 10  7  1  0
 5  a  y<-x  0
 0 10  7  1  0
 0  0  2  0  0

a는 모래가 이동하고 남은 양이며, 모래 이동 간의 소수점 이하의 수는 버린다.
이동한 모래는 기존 모래와 더해지며, 모래는 판을 벗어나 이동할 수 있고 이 경우 모래는 소멸된다.

토네이도가 소멸되었을 때, 판을 벗어난 모래의 양을 구하여라.

* 제약 사항:
N은 3이상 499이하의 홀수
모래의 양은 0이상 1,000이하
(N/2, N/2)칸의 모래의 양은 0

* 입력:
판의 크기 N
각각의 판에 대한 모래의 양 (N*N)

* 출력:
소멸된 모래의 양

* 풀이:
이동을 수치화 및 간편화 해야함
진행 방향에 따른 모래 이동을 정형화 해야함
1. 토네이도 진행
2. 모래 이동
3. 이동한 모래량 합산 및 소멸량 계산
